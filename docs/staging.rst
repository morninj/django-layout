Staging
=======

First, commit all local changes to the repository.

Add Staging Settings
--------------------

    $ cd /path/to/virtualenvs/project_name/src/project_name

Edit the following variables in ``fabfile.py``:

-  ``USER``
-  ``STAGING_SERVER``
-  ``PRIVATE_KEY_FILE``
-  ``VIRTUALENV_ROOT``
-  ``REPOSITORY_READ_ONLY``
